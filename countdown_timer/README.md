# Countdown Timer

**Difficulty Level:** Beginner (Est. 1 Hour)

## Project Description
Create a simple command-line countdown timer. The user enters a number of seconds, and the script counts down to zero, printing the remaining time each second. This project introduces the `time` module, specifically the `time.sleep()` function, within a `while` loop.

## Possible Folder Structure
```
countdown_timer/
└── main.py
```

## Inputs and Expected Outputs
- **Input:** `Enter seconds: 5`
- **Output:** `5...`, `4...`, `3...`, `2...`, `1...`, `BLAST OFF!`

## Learning Objectives
- Time module usage
- While loops
- Sleep functionality
- Timer implementation
