# Web Scraper with BeautifulSoup

**Difficulty Level:** Intermediate (Est. 2-3 Hours)

## Project Description
Build a scraper to get specific data from a web page. Use `requests` to get the HTML of a page (e.g., a "Hacker News" page) and `BeautifulSoup` to parse the HTML. Extract all the article titles and their links and print them to the console.

## Possible Folder Structure
```
web_scraper/
├── main.py
└── requirements.txt
```

## Inputs and Expected Outputs
- **Input:** (None, URL is hard-coded in the script)
- **Output:** A list of titles and URLs from the target page

## Learning Objectives
- Web scraping techniques
- HTML parsing with BeautifulSoup
- Data extraction from websites
- HTTP requests handling
