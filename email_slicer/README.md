# Email Slicer

**Difficulty Level:** Beginner (Est. 1 Hour)

## Project Description
A simple script that takes an email address from the user and "slices" it into its username and domain. This is another string manipulation exercise, focusing on finding the "@" symbol and using string splitting or indexing.

## Possible Folder Structure
```
email_slicer/
└── main.py
```

## Inputs and Expected Outputs
- **Input:** `Enter your email: user@example.com`
- **Output:** `Username: user`, `Domain: example.com`

## Learning Objectives
- String splitting
- String indexing
- Email parsing
- Data extraction
