# Password Generator

**Difficulty Level:** Beginner (Est. 1 Hour)

## Project Description
Write a script that generates a random password of a user-specified length. The password should include a mix of letters (upper and lower), numbers, and symbols. This project uses the `random` and `string` modules. It's a great exercise in `for` loops, list manipulation, and `random.shuffle()`.

## Possible Folder Structure
```
password_generator/
└── main.py
```

## Inputs and Expected Outputs
- **Input:** `Enter password length: 12`
- **Output:** `Your new password is: Gt8!#pW_9z$K`

## Learning Objectives
- Random module
- String module
- List manipulation
- For loops
- Password security concepts
